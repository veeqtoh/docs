---
title: "Introduction"
description: "Prompt Deck is a Laravel package for versioned, file-based AI prompt management with variable interpolation, performance tracking, and Laravel AI SDK integration."
---

Prompt Deck helps you organise your AI agent instructions as structured, version-controlled files. It provides variable interpolation, performance tracking, A/B testing, and optional seamless integration with the [Laravel AI SDK](https://laravel.com/ai) making it easy to iterate, compare, and activate prompt versions across your Laravel application.
